# Topic Reasearch with CrewAI and Streamlit

## Tools used 
Python 
CrewAI for Agentic Workflow
Streamlit for UI
Mistral AI as LLM 
SerperDevTool for Google Serper Search



## Run Streamlit app
```
conda create -n topic python=3.12

conda activate topic

make deps

streamlit run streamlit_app.py

```
